My Ubuntu Setting

I use Ubuntu 22.04 LTS with the GNOME desktop environment. My setup includes:

  • Oh My Zsh with custom themes and plugins
  • Tilix and GNOME Terminal for multi-tabs and split view
  • Custom VS Code settings synced via GitHub
  • dotfiles managed with GNU Stow
  • Favorite tools: htop, neofetch, tmux, and ranger

Feel free to check out my dotfiles repository on GitHub for more details.

apt installs

  • git: Version control system for tracking changes in source code.
  • htop, bpytop: Interactive process viewer for Unix systems.
  • neofetch: Displays system information in a visually appealing way.
  • caffeine: Prevents the system from going to sleep or activating the screensaver.
  • make: Utility for managing build automation.
  • vlc: Media player that supports various audio and video formats.
  • handbrake: Open-source transcoder for digital video and audio files.

Getting rid of snap

To remove snap and its associated packages, you can run the following commands:

sudo apt purge snapd
        sudo rm -rf /var/cache/snapd
        sudo rm -rf ~/snap

After running these commands, you can install your preferred applications using apt or other package managers.

dpkg installs

  • google-chrome-stable: Web browser from Google. Visit Google Chrome for more information.
  • code: Visual Studio Code editor. Visit Visual Studio Code for more information.

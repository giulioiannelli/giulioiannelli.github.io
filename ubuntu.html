My Ubuntu Setting

I use Ubuntu 22.04 LTS with the GNOME desktop environment. My setup includes:

  • Oh My Zsh with custom themes and plugins
  • Tilix and GNOME Terminal for multi-tabs and split view
  • Custom VS Code settings synced via GitHub
  • dotfiles managed with GNU Stow
  • Favorite tools: htop, neofetch, tmux, and ranger

Feel free to check out my dotfiles repository on GitHub for more details.

apt installs

  • git: Version control system for tracking changes in source code.
  • htop, bpytop: Interactive process viewer for Unix systems.
  • neofetch: Displays system information in a visually appealing way.
  • caffeine: Prevents the system from going to sleep or activating the screensaver.

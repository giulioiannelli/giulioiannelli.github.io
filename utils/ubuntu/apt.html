Essential Tools

  • git: Version control system for tracking changes in source code.
  • htop, bpytop: Interactive process viewer for Unix systems.
  • neofetch: Displays system information in a visually appealing way.
  • caffeine: Prevents the system from going to sleep or activating the screensaver.
  • curl: Command-line tool for transferring data with URLs.

Media Tools

  • vlc: Media player that supports various audio and video formats.
  • handbrake: Open-source transcoder for digital video and audio files.
  • ffmpeg: Complete solution to record, convert and stream audio and video.

Development Tools

  • make: Utility for managing build automation.
  • build-essential: Package containing reference headers and libraries for C/C++.
  • python3-pip: Package installer for Python.
  • nodejs: JavaScript runtime built on Chrome's V8 JavaScript engine.

Disk Space Analyzer

This script helps you identify what's taking up space on your system:

#!/bin/bash
echo "Top 10 directories by size:"
du -h /var /home /usr --max-depth=2 2>/dev/null | sort -hr | head -n 10

Quick Note Taking

A simple script to quickly create timestamped notes:

#!/bin/bash
# Quick note script
NOTES_DIR="$HOME/Documents/notes"
mkdir -p "$NOTES_DIR"
TIMESTAMP=$(date +"%Y-%m-%d_%H-%M-%S")
nano "$NOTES_DIR/note_$TIMESTAMP.txt"

Work Timer

A Pomodoro-style timer for productivity:

#!/bin/bash
# Pomodoro timer
work_minutes=25
break_minutes=5

echo "Starting a $work_minutes minute work session"
sleep ${work_minutes}m && notify-send "Break Time!" "Take a $break_minutes minute break" &
sleep $((work_minutes + break_minutes))m && notify-send "Back to Work!" "Start another session" &

Custom Aliases

Useful aliases to add to your .bashrc or .zshrc file:

# Quick navigation
alias ..='cd ..'
alias ...='cd ../..'
alias ....='cd ../../..'
alias home='cd ~'
alias dev='cd ~/Development'

# Enhanced commands
alias ls='ls --color=auto'
alias ll='ls -alF'
alias la='ls -A'
alias l='ls -CF'
alias cp='cp -i'
alias mv='mv -i'

# System commands
alias update='sudo apt update && sudo apt upgrade -y'
alias clean='sudo apt autoremove -y && sudo apt clean'
alias sysinfo='neofetch'
alias diskspace='du -h --max-depth=1 | sort -hr'

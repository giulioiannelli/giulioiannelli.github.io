Some basic settings

Here are some basic settings you can use with Gsettings on Ubuntu:

  • Change the default font: gsettings set org.gnome.desktop.interface font-name 'Sans 11'
  • Set the background color: gsettings set org.gnome.desktop.background primary-color '#000000'
  • Enable dark mode: gsettings set org.gnome.desktop.interface gtk-theme 'Adwaita-dark'
  • Adjust touchpad speed: gsettings set org.gnome.desktop.peripherals.touchpad speed 0.5
  • Check lock screen on suspend: gsettings set org.gnome.desktop.screensaver ubuntu-lock-on-suspend true
  • Disable lock screen on suspend: gsettings set org.gnome.desktop.screensaver ubuntu-lock-on-suspend false

Keyboard shortcuts

Here are some useful keyboard shortcuts for Ubuntu:

  • Super + A: Show the application menu
  • Super + T: Open a terminal
  • Super + L: Lock the screen
  • Alt + Tab: Switch between open applications

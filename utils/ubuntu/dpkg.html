To install packages using DPKG, you can use the following command:

sudo dpkg -i package_name.deb

Make sure to replace package_name.deb with the actual name of the package you want to install.

Web Browsers

  • google-chrome-stable: Web browser from Google. Visit Google Chrome for more information.
  • firefox: Mozilla Firefox browser. Though it comes pre-installed on Ubuntu, you might want to get the latest version from Mozilla's website.

Development Tools

  • code: Visual Studio Code editor. Visit Visual Studio Code for more information.
  • conda: Anaconda package manager for Python. Visit Anaconda for more information. In this case we need to run a .sh script so after downloading an installer such as Anaconda3-<version>-Linux-x86_64.sh just make it executable with 
    chmod +x Anaconda3-<version>-Linux-x86_64.sh
    and then run it with 
    ./Anaconda3-<version>-Linux-x86_64.sh
    If you want to install it in a specific directory, you can specify the path during the installation process, but at the end you would need to add the Anaconda path to your .bashrc file: 
    export PATH="/path/to/anaconda3/bin:$PATH"

Other Applications

  • slack-desktop: Team collaboration tool.
  • discord: Voice, video, and text communication service.
